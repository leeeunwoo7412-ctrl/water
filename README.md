# 💧 광광역시별 수돗물 화학 성분 및 물맛 분석 웹 서비스

당곡고등학교 과학 탐구 토론 및 자율 탐구 활동을 위해 제작된 Streamlit 기반 시뮬레이터 프로그램입니다. 
지질학적 특성에 따른 수질 화학 성분 데이터 분석과 의학적 영향, 그리고 미네랄 조절 정수 기계 설계 개념을 포함하고 있습니다.
